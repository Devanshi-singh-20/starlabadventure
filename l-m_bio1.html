Complete procedure for the Starch Test experiment

🌿 Aim

To test the presence of starch in a given food sample using iodine solution.

🔧 Materials Required

  • Food samples (potato, rice, apple, banana, onion)
  • Test tube
  • Dropper
  • Iodine solution
  • Beaker
  • Water
  • Filter paper & funnel

🔬 Procedure

  1. Crush the food sample using a mortar and pestle.
  2. Add a few drops of water to make a paste or solution.
  3. Filter the solution using filter paper to obtain clear extract.
  4. Pour the extract into a clean test tube.
  5. Add a few drops of iodine solution using a dropper.
  6. Observe the color change in the test tube.

🔍 Observation

If the color changes to blue-black, starch is present.

If there is no color change, starch is absent.
